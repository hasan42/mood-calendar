## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Frontend | Vanilla JS (ES6+) или React |
| Стили | CSS3 + CSS Variables (темы) |
| VK Integration | @vkontakte/vk-bridge |
| UI Kit | VKUI (опционально) или кастом |
| Charts | Chart.js или Canvas API |
| Хранение | VK Storage API + localStorage |

---

## Зависимости

```html
<!-- VK Bridge -->
<script src="https://unpkg.com/@vkontakte/vk-bridge/dist/browser.min.js"></script>

<!-- VK UI (опционально) -->
<link rel="stylesheet" href="https://unpkg.com/@vkontakte/vkui/dist/vkui.css">
<script src="https://unpkg.com/@vkontakte/vkui/dist/vkui.js"></script>

<!-- Chart.js (для статистики) -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
```
