# Календарь Настроений

Простое приложение для отслеживания настроения через цветовую маркировку дней.

## Быстрый старт

### Локальная разработка

1. Откройте `index.html` в браузере
   - Или используйте локальный сервер:
   ```bash
   npx serve .
   # или
   python -m http.server 8000
   ```

2. Для работы VK Bridge нужен HTTPS:
   ```bash
   npx serve --ssl
   ```

## Что реализовано:

### ✅ Готово:
- Отображение календаря на месяц
- Навигация по месяцам (← →)
- Выделение текущего дня
- 6 цветов настроений (пресеты)
- **Кастомные цвета** — добавление через "+" с названием
- Сохранение в localStorage (fallback)
- **VK Storage API** — полноценная интеграция
- **Статистика** — месяц/год, распределение цветов, проценты
- VK Bridge подключён (App ID: 54534829)
