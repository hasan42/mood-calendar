# Календарь Настроений

Приложение для отслеживания эмоционального состояния через цветовую маркировку дней. Работает как VK Mini App и веб-приложение.

🔗 **Живой демо:** https://hasan42.github.io/mood-calendar/  
📱 **VK Mini App:** https://vk.com/app54534829

---

## ✨ Возможности

### 🎨 Отслеживание настроения
- **6 пресетных цветов** с описаниями:
  - 🟢 **Отличное** — энергия, радость, продуктивность
  - 🟡 **Нейтральное** — спокойствие, обычный день  
  - 🔴 **Плохое** — стресс, раздражение, усталость
  - 🔵 **Грусть** — меланхолия, тоска, разочарование
  - 🟣 **Вдохновение** — творчество, эйфория, подъём
  - ⚫ **Очень плохое** — депрессия, тревога, выгорание
- **Кастомные цвета** — добавь свои настроения
- **Градиентные цвета** — современный визуальный стиль
- **Удаление настроения** — исправь выбор одним кликом

### 📅 Календарь
- Месячный просмотр с цветными маркерами
- Навигация: ← → или клик по месяцу
- Кнопка "Сегодня" — мгновенный возврат к текущей дате
- Плавающая кнопка "+" — быстрое добавление настроения
- Плавные анимации переключения месяцев

### 🔥 Годовой Heatmap
- Визуализация всего года в сетке 12×31
- Навигация по годам
- Клик на день — переход к дате в календаре
- Легенда цветов

### 📊 Статистика
- Распределение настроений за месяц/год
- Процентное соотношение с визуальными барами
- Подсчёт дней с записями

### 💾 Экспорт и импорт
- 💾 **Экспорт** — скачать все данные в JSON
- 📥 **Импорт** — восстановить из резервной копии  
- 📤 **Поделиться** — пост в VK со статистикой

### 🎨 Темы оформления
- ☀️ Светлая тема
- 🌙 Тёмная тема
- 🌓 Авто (по системным настройкам)
- Переключатель в шапке приложения

### 📱 Для VK Mini Apps
- Интеграция VK Bridge API
- Хранение данных в VK Storage (синхронизация между устройствами)
- Fallback на localStorage
- Адаптивный дизайн для мобильных

## 🚀 Быстрый старт

### Локальная разработка

```bash
# Клонировать репозиторий
git clone https://github.com/hasan42/mood-calendar.git
cd mood-calendar

# Запустить локальный сервер
npx serve .
# или
python -m http.server 8000
```

Откройте http://localhost:8000

### Для VK Mini Apps

```bash
# Для работы VK Bridge нужен HTTPS
npx serve --ssl
```
