# Product Requirements Document (PRD)
## Календарь Настроений (Mood Calendar)

### Цель
Простое и интуитивное приложение для отслеживания эмоционального состояния через цветовую маркировку дней.

### Основной функционал

#### 1. Выбор цвета для дня
- Предустановленные цвета настроений:
  - 🟢 Зелёный — отличное настроение, продуктивность
  - 🟡 Жёлтый — нейтральное, спокойное состояние
  - 🔴 Красный — плохое настроение, стресс, усталость
  - 🔵 Синий — грусть, меланхолия
  - 🟣 Фиолетовый — творческий подъём, вдохновение
  - ⚫ Чёрный — очень плохой день (депрессия, выгорание)
- **Кастомные цвета**: пользователь может добавить свой цвет с названием и описанием

#### 2. Просмотр календаря
- Месячный вид календаря с цветными маркерами
- Годовой обзор (heatmap года)
- Навигация по месяцам/годам
- Тап на день — открытие деталей/редактирование

#### 3. Хранение данных
- Локальное хранение в VK Mini Apps (VK Storage API)
- Экспорт данных (JSON/CSV)
- Возможно: синхронизация между устройствами (VK ID)

#### 4. Статистика
- Самые частые настроения за период
- Тренды (улучшение/ухудшение)
- Процентное соотношение цветов

### Платформа
- VK Mini Apps (HTML5/JS)
- Десктоп и мобильная версия VK

### Дизайн
- Минималистичный, clean UI
- Тёмная и светлая тема (автоопределение)
- Плавные анимации
- Material Design или Apple-like aesthetics
